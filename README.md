# ♻️ Recycling-Classification-Page
분리수거 재활용 체크 사이트

Google Teachable Machine을 활용한 재활용 분류 웹 애플리케이션

## 🌟 주요 기능

- 📷 **실시간 카메라 촬영**: 모바일 카메라로 재활용품을 촬영하여 즉시 분류
- 📁 **이미지 업로드**: 갤러리에서 사진을 선택하여 분류
- 🤖 **AI 기반 분류**: Teachable Machine 모델을 사용한 정확한 분류
- 📱 **모바일 최적화**: 휴대폰에서 사용하기 편리한 반응형 디자인
- 🇰🇷 **한국어 지원**: 완전한 한국어 인터페이스

## 🚀 배포 방법

### Vercel 배포

1. Vercel 계정으로 로그인
2. GitHub 저장소 연결
3. 자동 배포 시작

또는 Vercel CLI 사용:

```bash
npm install -g vercel
vercel
```

### 로컬 실행

```bash
npm start
```

브라우저에서 `http://localhost:3000` 접속

## 🛠️ 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **AI Model**: Google Teachable Machine
- **ML Library**: TensorFlow.js, Teachable Machine Library
- **Deployment**: Vercel

## 📝 사용 방법

1. 웹사이트 접속
2. "카메라로 촬영" 또는 "사진 업로드" 선택
3. 재활용품 촬영/업로드
4. AI가 자동으로 분류 결과 표시
5. 분류 확률과 함께 결과 확인
